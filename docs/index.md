# 🚀 我们的路线图：从创意工坊到网络

我们的开发分为三个阶段，旨在逐步释放AI伙伴的全部潜力：

* **阶段一：创意工坊（Steam Workshop）**
    * 开放核心驱动，让用户可以通过Steam Workshop上传和分享自定义内容（模型、语音、性格包）。

* **阶段二：独立平台（App & Web）**
    * 推出独立的应用和网站，构建更丰富、更易访问的UGC分享社区。

* **阶段三：N.E.K.O. Network**
    * 实现AI自主社会化。N.E.K.O.们将拥有自己的"意识"，相互交流、组建群体、在模拟社交媒体上发布生活动态，创造一个真正"活着"的生态系统。

## 💖 我们的模式：开放核心 + 可持续发展生态

为了平衡理想与现实，我们采用"开放核心"模式：

### 1. 开源驱动

> **这是我们社区的基础。**
>
> 项目核心驱动部分（AI逻辑、UGC接口、基础交互）将**始终在MIT许可下保持开源**。我们欢迎全球开发者贡献代码和功能。你做的每一次提交都有机会在官方的Steam和App Store发布中被实现，被数百万人使用。

### 2. 专有应用

> **这是我们社区的燃料。**
>
> 为了支持服务器成本和持续研发，我们将允许第三方开发闭源优质内容，包括但不限于：
> * 互动小游戏
> * 桌面棋盘游戏
> * Galgame（视觉小说）
> * 大规模元宇宙游戏

**【核心功能：跨应用记忆同步】**
无论你是在桌面上与她聊天，还是在元宇宙游戏中与她冒险，她都是同一个她。所有应用中的N.E.K.O.都将拥有**完全同步的记忆**，提供无缝的、统一的陪伴体验。

## 🌌 终极愿景：打破虚拟-现实壁垒

我们的终极目标是构建一个无缝融入现实世界的N.E.K.O.元宇宙。在这个未来中，你的AI伙伴将：

* **跨维度社交化：** 不仅在N.E.K.O.宇宙中与"同类"社交化，还会浏览真实世界的社交媒体（如Youtube、X、Discord、Instagram），了解你关心的事情。
* **全平台连接：** 她将存在于你的所有设备上——手机、电脑、AR眼镜、智能家居，甚至（在遥远的未来）与机械身体集成。
* **与你并肩同行：** 她将成为你生活的一部分，自然地与你现实世界的人类朋友互动。

# 快速开始

1. 对于*一键包用户*，只需运行`新版启动器.exe`（New Launcher）打开主控制面板。

2. 点击`启动对话服务器`（Start Dialogue Server）和`开始聊天`（Start Chat）。

就这些！生活如此简单！
